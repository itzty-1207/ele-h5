1. 交互方式
- 前端http请求是跟后端交互的重要手段和形式
- 基于Promise网络请求库
- 同构，同一套代码可运行在浏览器和Nodejs中
- 在服务端使用原生node的http模块，在客户端使用XHR

2. 特性
- 支持Promise API
- 拦截请求和响应
- 取消请求
- 自动转换JSON数据

3. http状态码
- 表明http请求是否成功完成
- 1信息响应
- 2成功响应
- 3重定向消息
- 4客户端错误响应
- 5服务端错误响应

4. 请求拦截
- 在请求或相应被then或catch处理前进行拦截

5. 业务状态码
- 扩展请求的状态
- 成功、失败
